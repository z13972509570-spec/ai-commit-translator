## 核心模块

### 1. Commit Model (commit.py)
- `CommitType`: 12 种 commit 类型枚举
- `CommitMessage`: 结构化 commit 数据模型
- `detect_type()`: 基于关键词推断类型

### 2. Translator (translator.py)
- 支持 OpenAI / Anthropic / Ollama
- 懒加载 LLM 客户端
- 降级规则引擎

### 3. CLI (cli/main.py)
- `translate`: 主翻译命令
- `install-hook`: Git hooks 集成
- `uninstall-hook`: 卸载 hooks

## 支持的 Commit 类型

| 类型 | 说明 | 关键词 |
|------|------|--------|
| feat | 新功能 | 新增、添加、增加 |
| fix | 缺陷修复 | 修复、bug、错误 |
| docs | 文档 | 文档、readme |
| style | 样式 | 样式、格式 |
| refactor | 重构 | 重构、优化代码 |
| perf | 性能 | 性能、优化 |
| test | 测试 | 测试 |
| chore | 杂项 | 工具、配置、依赖 |
| ci | CI/CD | ci、cd、pipeline |
| build | 构建 | 构建、编译 |
| revert | 回滚 | 回滚 |
| hotfix | 紧急修复 | hotfix、紧急 |
